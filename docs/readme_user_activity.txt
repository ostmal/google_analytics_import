*** Скрипт для работы с функционалом User Activity Google Analytics-3 ***

В работе скрипта используется API ver.4  

Файлы запуска: 
ua_stage1_get_clientids.py
ua_stage2_get_user_activity.py

В папке "etc" содержатся конфигурационный файл:
ga_ua_config.yml	- общий файл конфигурации

Скрипт работает в 2 этапа:
1. Считывание из GA-3 массива "clientid-transactionsиз GA"
2. вызов функционала User Activity GA и сохранение результатов по каждому CLIENT_ID 

В конфигурационном файле задаются:
- Параметры доступа к GA
- Параметры доступа к БД
- Общие параметры
- Параметры этапа 1
- Параметры этапа 1


Возможности этапа 1:
- Запись данных в БД и/или CSV-файл

Возможности этапа 2:
- Считывание данных из БД или CSV-файла
- Запись данных в БД и/или CSV-файл
- Запрет записи в итоговый файл CustomDimensions
- Возможность указать перечень CustomDimensions для записи в итоговый файл
